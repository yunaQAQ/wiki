新创建的 Ubuntu 系统服务器，需要的操作。 (注：从 16.04 开始可以 apt 替代 apt-get 使用APT简化命令行)

更新包

sudo apt update

创建管理员账号

sudo adduser yuna --ingroup sudo
cd ~

常用包

sudo apt install -y git-core
sudo apt install -y nginx-core
